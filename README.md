This website uses:
- React with Vite
- Tailwind CSS
- JavaScript
- EmailJS

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/francescogiorgetti-dev/portfolio.git
cd your-repo-name
npm install
npm run dev
```
